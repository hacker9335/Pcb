# Gerber render
its an app tht takes in gerber files and visualizes them

app tht runs on streamlit uses the py gerber library to visualize the gerber files given by the user the visualized files are stacked together to form one pcb.

made the app for the shipyard week one challenge and got the idea as i was working on designing a pcb when the challenge ws given

playable link : https://gerber.streamlit.app/
